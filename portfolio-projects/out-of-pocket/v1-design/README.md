# V1 Design Gallery - Out of Pocket

A walkthrough of the original v1 interface design for Out of Pocket: Phone Gone Wild, covering every screen from onboarding through gameplay. These designs established the visual language, gesture system, and game flow that shipped in the first App Store release.

## Gallery Contents

- **33 Screenshots** organized by game mode and feature
- **Sections:** Home Screen, Tutorial (14 gestures), Classic Gameplay, Memory Mode, Pass & Play, Results & High Scores, Settings
